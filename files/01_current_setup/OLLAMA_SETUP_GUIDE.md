# 🚀 OLLAMA SETUP GUIDE (100% Free Local LLM)

## What is Ollama?

Ollama is a **free, local AI model runner** that works entirely on your computer. No API keys, no monthly bills, no vendor lock-in.

Think of it as: "Run ChatGPT on your computer, for free"

---

## ✅ INSTALLATION

### Windows

**Option 1: Installer (Easiest)**
1. Download: https://ollama.ai/download/windows
2. Run installer
3. Restart computer
4. Done


|-------|------|-------|---------|----------|
| **neural-chat** | 4GB | ⚡⚡⚡ (1-2 min) | Good | YouTube scripts, daily use |

## 🔧 RUNNING OLLAMA

### Start the Server

```bash
# Open terminal and run:
ollama serve

# Output should show:
# listening on 127.0.0.1:11434
```

**Keep this terminal window open!** (It's your LLM server)

### Verify It's Working

In a **NEW terminal window**:

```bash
# Test the connection
curl http://localhost:11434/api/tags

# Should return list of your models
```

## ⚙️ OPTIMIZATION TIPS

### For Speed (If Videos Take Too Long)

**Reduce model precision (faster, slight quality loss):**

```bash
# Instead of:
ollama pull mistral

# Use quantized version (smaller, faster):
ollama pull mistral:7b-instruct-q4_K_M
# or
ollama pull neural-chat:7b-v3-q4_K_M
```

Quantized = Compressed model = ~2x faster, ~1% quality loss

### For Quality (If Scripts Are Poor)

```bash
# Use larger model:
ollama pull dolphin-mixtral
# or
ollama pull llama2:13b
```

More parameters = Better quality, but slower

### Use GPU if You Have It

**NVIDIA GPU (CUDA):**
Ollama auto-detects NVIDIA GPUs. Just have:
- NVIDIA drivers installed
- CUDA 11.5+ (usually auto-installed with drivers)

Check: `nvidia-smi` (should show GPU)

**AMD GPU:**
Ollama supports ROCm, but requires setup. See: https://ollama.ai/docs

**Mac with Apple Silicon:**
Auto-optimized, very fast!

## 🛠️ TROUBLESHOOTING

### Problem: "Port 11434 already in use"

```
Ollama tried to start but another process is using the port

Fix:
1. Close other Ollama instances
2. Or change port: ollama serve --listen 127.0.0.1:8000
3. Update server_ollama.py OLLAMA_URL = "http://localhost:8000"
```

### Problem: "Model download stuck"

```
If stuck at 50% for 10+ minutes:

Fix:
1. Press Ctrl+C to stop
2. Try again: ollama pull mistral
3. Or use faster download: ollama pull neural-chat

Common: Connection issues to Ollama servers
Try on different network or later
```

### Problem: "Video generation very slow (10+ mins)"

```
Likely: Using CPU instead of GPU

Check:
1. Do you have GPU? nvidia-smi
2. Is it detected? Check Ollama startup message
   Should say: "Selected GPU: NVIDIA..." or similar

Fix:
1. Make sure NVIDIA drivers updated
2. Use smaller model: ollama pull neural-chat:7b-v3-q4_K_M
3. Or increase system RAM for faster CPU processing
```

### Problem: "Ollama keeps crashing"

```
Fix:
1. Close Ollama
2. Clear cache: rm -rf ~/.ollama (Mac/Linux) or clear %USERPROFILE%\.ollama (Windows)
3. Restart Ollama: ollama serve
4. Re-download model: ollama pull mistral
```

### Problem: "Scripts are bad quality"

```
Possible issues:
1. Model too small
   Fix: ollama pull llama2:13b (better quality)
   
2. Prompt not clear
   Fix: Check server_ollama.py line ~180, improve prompts
   
3. Temperature too high (too random)
   Fix: Change temperature from 0.7 to 0.5 in server_ollama.py line ~200
```
